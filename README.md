# cprofile-analyzer

Standalone analyzer for Python `cProfile` `.prof` files (cumulative/self time, callers/callees, text/JSON).

## Install

```bash
pip install -e .
cprofile-analyzer profile.prof
```

## Origin

Extracted from PyKotor `helper_scripts/python/analyze_profile.py`.
